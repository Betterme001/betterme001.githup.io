4. 动名词

   1. 动名词可以充当的成分：主宾表定，宾补

      1. 作主语，
         1. 句首
         2. 特定句型
            1. It is no good/use/
            2. It is a waste of time
            3. There is no doing sth 不可能。。
            4. It helped doing sth
            5. it is hard enough: doing/todo 

      2. 作宾语（动词宾语，介词宾语）
         1. 动名词作并宾语的动词有哪些

      3. 作表语: seeing is believing

      4. 作定语表示用途和功能。reading room
      5. 作宾补：I saw him **running** in the park.

      6. 特定句型、

         1. have difficulty/ a hard time/problem/trouble doing
         1. 情不自禁：can't help doing/can't hold back from
         1. 值得琢磨事 sth is worth doing/it is worth doing sth， sth be worth doing：The book is worth reading

   2. 动名词复合结构

      1. 物主代词（例my, his, her等) ,
      2. 所有格名词（例Will, Tom's 等）+doing

      2. 动名词复合结构作宾语时 用普通格 和 所有格 均可


      3. 动名词的被动式: being/主动表被动

         1. being +被动式

         2. 主动表被动：need/wants/deserve


   4. 动名词完成式：动名词的动作在谓语动作之前发生

5. 不定式
      1. 充当的成分：主宾表定状补
         1. 主语
            1. It is adj/n to do..
                     1. 不定式的逻辑主语for/of
            2. It. 动词 todo： It takes two  to tango.
         2. 宾语：
                  1. 动词+todo
                  2. 动词+疑问词when/how/which等）+to do
                  3. 动词+it(形式宾语）+adj（宾补）+to do（宾语）
         3. 不定式作定语
                  1. 主谓关系: man/person
                  2. 动宾关系: clothes/letter/kids/man
                  3. 动状关系:way/time/reason
                  4. 同位关系 chance/opportunity/dream/attempt/ability/paln/promise/anxiety/decision/need
                           1. 不定式与所修饰的名词指的是一回事。(不定式为解释前面名词具体内容）chance/opportunity/dream
                  5. 某些动词或形容词后可接不定式时，它相应的同根名词也常用不定式做定语。attempt/ability/paln/promise/anxiety/decision/need
         4. 不定式作状语
              1. 目的状语
              2. 结果状语
                   1. so... as to do
                   2. such... as to do
                   3. enough to do
                   4. only to do 示出乎意料的结果，而doing 表必然结果。
                   5. too... to do 太。。不能怎么
              3. 原因状语
                   1. Sb be +adj（情感） to do/have done… 做了某事而产生什么情感
              4. 条件状语：A man would be blind not to see that.
              5. 方式状语： sb/sth+（be）+adj+to do, to do作状语修饰adj
                    1. sb/sth is adj + to do， adj 作表语，to do修饰adj，to do和主语形成动宾关系，
                       1. He is easy to get along with. 他很容易相处。
                    2. v +sb/sth adj+ to do ，adj 作宾补，to do修饰adj，to do和宾语形成动宾关系，
                       1. I found the car comfortable to ride in. 我觉得这种车很好坐。
         5. 作补语：某些动词要省略不定式
              1. 感觉动词：see, watch, observe, hear, notice, feel
              2. 使役动词：let, make, have
      2. 不带to 的不定式（to do->do)
         1. 在口语中，动词原形come和go后可接不带to的不定式。 
         2. 在why引起的一些问句中，疑问词直接跟动词原形或not+动词原形。
         3. 在had better, had best, would rather, would rather...than, would sooner, would sooner...than,cannot but, cannot choose but, cannot help but等结构后直接跟动词原形或not+动词原形。
         4. 如不定式前有行为动词do，那么在表语从句中的不定式和介词except或but之后的不定式可不带to。
         5. 在两个动词不定式并列使用时，为了避免重复，后面的不定式符号to可省略。
         6. 在help之后，既可用带to的不定式也可用不带to的不定式。
         7. 在一些固定搭配中用不带to的不定式。
         8. 在感官动词see, hear, watch, feel, notice以及使役动词make,let, have等后作宾语补足语的动词不定式不带to。
      3. (不定式符号)to的单独使用(to do->to)
         1. 为了避免重复上文中出现过的动词，可以用不定式符号to来代替上文中出现过的不定式结构。
         2. 在助动词或情态动词之后，如begoing to, used to, have to, ought to, be able to, be about to等。
         3. 在want,decide, like, love, hope, wish, mean, refuse, try等动词之后。
         4. 在做宾语补足语的ask, tell, order, advise, persuade, warn, wish, permit, allow, forbid等动词之后
         5. 在对话的答语中的happy,glad, eager, anxious, willing, ready, pleased, afraid等形容词之后

6. 分词

   1. 分词的思维：
         1. 现在分词：主动进行，正在发生
         2. 过去分词：被动进行，已经完成

   2. 分词可以做那些成分：作定语，状语
   3. 非谓语动词作后置定语的三种被动结构
         1. 将来的动作：to be done
         2. 正在进行时:   being done
         3. 已经完成的： done

   4. 分词形容词：
         1. 名词/副词+分词
         2. 形容词+名词ed
         3. 数次+名词ed

7. 介词

   1. 三个功能：

         1. 表示动作的方向，范围和程度
         2. [胶水词]，连接不同的名词
         3. 与BE动词连用，代替动词

   2. 江南四大介词 on/at/to/in

         1. on 在...上
               1. 上车/船/台
               2. 集中...上
               3.  临界、接近、接壤，后面接“线状”的或者“带状”的东西
               4. 关于，就某一问题
               5. on与be 连用作表语，表状态式

         2. in 在...里
               1. 在某人的身上， 品质/气质/才华

         3. at 表示一个“点”，狭小的范围，精准的概念
            1. 时间点
            2. 地点
            3. 价格，速度
            4. 冲/朝着..(狭小的范围，精准的概念) @sb

      4. to：到；朝向；到达
